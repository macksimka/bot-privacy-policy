Privacy Policy Last updated: January 2026 1. Introduction This Privacy Policy describes how the Discord bot (the “Bot”) collects, uses, and stores data when used in Discord servers. The Bot is a moderation and server management tool designed to help server administrators maintain a safe and organized environment. 2. Data the Bot Collects The Bot collects and processes the following data only when necessary for moderation purposes: Discord user IDs (for moderation actions) Server (guild) IDs Channel IDs Message content only for real-time moderation Stored violations (rule breaks) Configured word filters (prohibited words list) Configuration data such as enabled channels and servers 3. Message Content Message content is only processed in real time Message content is not permanently stored Message content is not shared with third parties Message content is not used to train AI or machine learning models 4. Stored Data The following data may be stored off-platform in a database: User IDs associated with violations Violation history (timestamps and type of violation) Word filter configuration Server and channel configuration No sensitive personal data is stored. 5. Data Usage Collected data is used exclusively for: Automated moderation Enforcement of server rules Logging moderation actions Server configuration management 6. Data Retention Data is stored only as long as necessary for moderation purposes and can be removed upon request by a server administrator. 7. User Control Server administrators can: Disable the Bot at any time Remove stored data by removing the Bot from the server Configure which channels and features are active 8. Third Parties The Bot does not sell, share, or distribute collected data to third parties. 9. Changes This Privacy Policy may be updated. Continued use of the Bot constitutes acceptance of the updated policy. 10. Contact For questions or data removal requests, please contact the bot developer via Discord.
